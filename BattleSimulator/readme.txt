SETUP MODE
Play Button: 
Left Click - loads units and goes to BATTLE MODE.
Right Click - loads units and simulates 100 battles and outputs win/tie/loss ratio.
Load Button: loads saved unit information in setup.
Save Button: saves unit information.

Left clicking unit cells to select that cell. You can assign no job (no unit), or a job to a cell. A cell with a unit/job can be configured which skills it has at each row.

BATTLE MODE
left click goes to next turn
escape returns to setup mode

you can see the name of the status effect by hovering over their colored box. The number in the box is the number of stacks a status effect has. The number below the box is the expiration timer if the status effect has one.
